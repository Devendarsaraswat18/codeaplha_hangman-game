#  Hangman Game

##  Project Description
This project is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

The player guesses a randomly selected word one letter at a time. The game ends when the player guesses the word correctly or uses all 6 incorrect attempts.



##  Features

- Random word selection
- 5 predefined words
- Maximum 6 wrong guesses
- Input validation
- Win/Lose messages
- Console-based interface

##  Technologies Used

- Python 3


##  Concepts Used

- Random Module
- Lists
- Strings
- While Loop
- If-Else Statements
- User Input


##  How to Run

bash
python hangman.py
